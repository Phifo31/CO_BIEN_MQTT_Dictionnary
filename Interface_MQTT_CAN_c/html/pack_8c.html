Detailed Description

Conversion des données entre format JSON et trame CAN (8 octets).

Ce module contient les fonctions responsables de la transformation :

  • du format JSON (utilisé par MQTT) vers le format binaire CAN (pack)
  • et inversement (unpack).

Il s’appuie sur la table de conversion chargée depuis conversion.json, qui indique le type de chaque champ (int, bool, hex, enum…).

Convertit un objet JSON en tableau de 8 octets CAN.

Cette fonction transforme chaque champ du JSON (int, bool, hex, etc.) selon le type défini dans la table.
Les valeurs sont ensuite placées dans le tableau de 8 octets à envoyer sur le bus CAN.
